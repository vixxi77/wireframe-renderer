## DEPENDENCIES
- SDL2<br>
Linux:
```bash
sudo apt install libsdl2-dev
```

## BUILDING & RUNNING
To build the project:
```bash
cd wireframe-renderer/
make
```
To run the project:
```bash
make run
```
